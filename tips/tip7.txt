Below is the most useful way to interpret your request: **free, hosted, API-callable LLM endpoints for coding**, ranked mostly by coding strength, then by practical free API throughput.

“Connections/hour” usually is not how LLM APIs publish limits, so I’m using:

**RPH = requests per hour = RPM × 60 burst capacity**, unless the provider publishes a direct RPH.
Daily caps, token caps, concurrency caps, free credits, or “neuron” quotas may throttle you before the RPM/RPH ceiling.

This is current to **May 6, 2026**. Free model availability changes quickly, especially OpenRouter `:free` models and preview endpoints.

## API key setup + example calls

### 1) Google Gemini API

Setup URL: `https://aistudio.google.com/app/apikey`

Steps:

1. Sign in to Google AI Studio.
2. Open **API Keys**.
3. Create a key.
4. Export it locally:

```bash
export GEMINI_API_KEY="YOUR_KEY"
```

Use for ranks **1, 5, 15**.

```bash
MODEL="gemini-3-flash-preview"   # or gemini-2.5-pro, gemini-2.5-flash

curl "https://generativelanguage.googleapis.com/v1beta/models/${MODEL}:generateContent?key=${GEMINI_API_KEY}" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "parts": [
          {
            "text": "You are an expert coding agent. Implement a production-grade TypeScript rate limiter with Redis, sliding-window semantics, tests, and edge-case explanations."
          }
        ]
      }
    ]
  }'
```

### 2) Cerebras Cloud

Setup URL: `https://cloud.cerebras.ai`

Steps:

1. Create a Cerebras Cloud account.
2. Open **API Keys** in the left navigation.
3. Create a key.
4. Export it:

```bash
export CEREBRAS_API_KEY="YOUR_KEY"
```

Use for ranks **2, 3, 7**.

```bash
MODEL="gpt-oss-120b"   # or zai-glm-4.7, qwen-3-235b-a22b-instruct-2507

curl https://api.cerebras.ai/v1/chat/completions \
  -H "Authorization: Bearer ${CEREBRAS_API_KEY}" \
  -H "Content-Type: application/json" \
  -d "{
    \"model\": \"${MODEL}\",
    \"messages\": [
      {
        \"role\": \"user\",
        \"content\": \"Write a Go worker pool with cancellation, backpressure, retries, metrics hooks, and table-driven tests.\"
      }
    ],
    \"max_completion_tokens\": 2048
  }"
```

### 3) Cloudflare Workers AI

Setup URL: `https://dash.cloudflare.com/?to=/:account/workers-ai`

Steps:

1. Sign in to Cloudflare.
2. Open **Workers AI**.
3. Choose **Use REST API**.
4. Create a **Workers AI API Token**.
5. Copy your **Account ID**.
6. Export both:

```bash
export CF_ACCOUNT_ID="YOUR_ACCOUNT_ID"
export CF_API_TOKEN="YOUR_TOKEN"
```

Use for ranks **4, 14, 19**.

```bash
MODEL="@cf/moonshotai/kimi-k2.6"   # or @cf/openai/gpt-oss-120b, @cf/qwen/qwen2.5-coder-32b-instruct

curl "https://api.cloudflare.com/client/v4/accounts/${CF_ACCOUNT_ID}/ai/run/${MODEL}" \
  -H "Authorization: Bearer ${CF_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {
        "role": "user",
        "content": "Build a Python FastAPI upload service with checksum validation, S3 storage, structured logging, and pytest tests."
      }
    ]
  }'
```

## Best practical free stack for coding

For strongest coding output, start with:

1. **Google `gemini-3-flash-preview`**
2. **Google `gemini-2.5-pro`**
3. **Cerebras `zai-glm-4.7`**
4. **Cerebras `gpt-oss-120b`**
5. **OpenRouter `qwen/qwen3-coder:free`**

For highest free request throughput, try:

1. **Cloudflare Workers AI** — 300 RPM default, but neuron-capped.
2. **Groq `qwen/qwen3-32b`** — 60 RPM, 1,000 RPD.
3. **Groq `openai/gpt-oss-120b`** — 30 RPM, 1,000 RPD.
4. **Cerebras `gpt-oss-120b`** — 30 RPM, 900 RPH, 14,400 RPD.
5. **OpenRouter free models** — 20 RPM, but daily free cap applies.

Do not paste private repositories, secrets, customer data, `.env` files, production credentials, or proprietary source into free tiers until you verify each provider’s data-use and retention terms for your exact account plan.
