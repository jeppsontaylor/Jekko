As of **May 6, 2026**, here is the strongest free hosted-API list I could verify from public docs. I’m ranking **model × host route**, not just model names, because the same model can be excellent on one host and nearly unusable on another due to free throttles.

**Important reading of the limits:** “RPH” below means API requests/hour. When a provider has both RPM and RPD, the **daily cap usually wins**. For example, 30 RPM looks like 1,800 RPH, but if the free tier is 1,000 RPD, you cannot sustain 1,800 requests in an hour for long.

## Best provider stack to wire into a coding agent

Use this order for best results:

```text
1. Google Gemini: gemini-3-flash-preview
2. Google Gemini: gemini-2.5-pro
3. NVIDIA NIM: qwen/qwen3-coder-480b-a35b-instruct
4. NVIDIA NIM: deepseek-ai/deepseek-v4-pro
5. NVIDIA NIM: moonshotai/kimi-k2-thinking
6. Cerebras: qwen-3-235b-a22b-instruct-2507
7. Cerebras: gpt-oss-120b
8. Groq: openai/gpt-oss-120b
9. Cloudflare Workers AI: @cf/moonshotai/kimi-k2.6
10. OpenRouter: qwen/qwen3-coder:free
```

For **highest documented free request volume**, Cerebras is cleaner than most: **900 RPH / 14,400 RPD** for Qwen3-235B and GPT-OSS-120B. For **highest published RPM**, Cloudflare’s text-generation default is **300 RPM**, but the free **10,000-neuron/day** allocation usually binds first. ([Cerebras Inference][9])

# API key setup and example calls

## 1. Google Gemini API

Google’s API keys are created in Google AI Studio; Google says Gemini rate limits are RPM/TPM/RPD and active limits are visible in AI Studio rather than published as one universal free RPM/RPH table. ([Google AI for Developers][14])

Setup URL:

```text
https://aistudio.google.com/app/apikey
```

Environment variable:

```bash
export GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
```

Example coding call:

````bash
curl "https://generativelanguage.googleapis.com/v1beta/models/gemini-3-flash-preview:generateContent" \
  -H "x-goog-api-key: $GEMINI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "parts": [
          {
            "text": "You are a senior coding assistant. Review this Python function for bugs, edge cases, and performance issues:\n\n```python\ndef average(xs):\n    return sum(xs) / len(xs)\n```"
          }
        ]
      }
    ]
  }'
````

Swap the model path for:

```text
gemini-2.5-pro
gemini-2.5-flash
```

## 3. OpenRouter

OpenRouter free models use `:free` model IDs. OpenRouter documents **20 requests/minute** for free models and a daily cap of **50 requests/day** for accounts with less than $10 lifetime credits, or **1,000/day** after at least $10 credits purchased. ([OpenRouter][7])

Setup URL:

```text
https://openrouter.ai/settings/keys
```

Environment variable:

```bash
export OPENROUTER_API_KEY="YOUR_OPENROUTER_API_KEY"
```

```bash
curl https://openrouter.ai/api/v1/chat/completions \
```

## 8. GitHub Models

GitHub Models provides no-cost rate-limited API access to supported models. Programmatic API use requires a GitHub token with model access; GitHub’s quickstart shows the `https://models.github.ai/inference/chat/completions` endpoint and `openai/gpt-4.1` as an example model. ([GitHub Docs][12])

Setup URLs:

```text
https://github.com/settings/personal-access-tokens
https://github.com/marketplace/models
```

Create a token with **Models: read** permission, then:

```bash
export GITHUB_TOKEN="YOUR_GITHUB_TOKEN"
```

```bash
curl -L https://models.github.ai/inference/chat/completions \
  -H "Authorization: Bearer $GITHUB_TOKEN" \
  -H "Accept: application/vnd.github+json" \
  -H "Content-Type: application/json" \
```

# Practical recommendation

For a coding tool, I would wire providers in this exact pattern:

```text
Quality-first primary:
  Gemini 3 Flash Preview
  Gemini 2.5 Pro
  NVIDIA Qwen3-Coder
  NVIDIA DeepSeek V4 Pro
  NVIDIA Devstral 2

High-volume documented fallback:
  Cerebras Qwen3-235B
  Cerebras GPT-OSS-120B
  Groq GPT-OSS-120B
  Groq Qwen3-32B

Free-router fallback:
  OpenRouter qwen/qwen3-coder:free
  Kilo x-ai/grok-code-fast-1:optimized:free
  Cloudflare Kimi K2.6 / GPT-OSS-120B / Nemotron
```

For proprietary code, I would be careful with free/trial endpoints. Some free routes explicitly warn that prompts and outputs may be logged, and several providers classify free APIs as evaluation or development use rather than production use.
