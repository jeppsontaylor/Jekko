As of **May 6, 2026**, here’s the usable list. I’m counting providers that let you make **hosted API calls for $0 upfront**. Some are truly ongoing/free-rate-limited; some are **monthly credits**, **trial credits**, or **free preview**. Also, most providers publish **RPM/RPD/TPM**, not “connections/hour,” so I compute **RPH = RPM × 60** when possible, but **daily caps often dominate**.

## API call examples

### 1. OpenAI-compatible providers

This works for **OpenRouter, Groq, Cerebras, Mistral, SambaNova, Hugging Face Router, NVIDIA NIM, Vercel AI Gateway, Fireworks, Alibaba Qwen, and GitHub Models** with the right base URL and model id.

```bash
export API_KEY="YOUR_KEY"
export BASE_URL="https://openrouter.ai/api/v1"
export MODEL="qwen/qwen3-coder:free"

curl "$BASE_URL/chat/completions" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "'"$MODEL"'",
    "messages": [
      {
        "role": "system",
        "content": "You are an expert coding assistant. Return concise, correct code."
      },
      {
        "role": "user",
        "content": "Write a TypeScript async debounce utility with Vitest tests."
      }
    ],
    "temperature": 0.2
  }'
```

Use this base URL table:

| Provider            | `BASE_URL`                                               | Example `MODEL`                                  |
| ------------------- | -------------------------------------------------------- | ------------------------------------------------ |
| OpenRouter          | `https://openrouter.ai/api/v1`                           | `qwen/qwen3-coder:free`                          |
| Groq                | `https://api.groq.com/openai/v1`                         | `qwen/qwen3-32b`                                 |
| Cerebras            | `https://api.cerebras.ai/v1`                             | `gpt-oss-120b`                                   |
| Mistral             | `https://api.mistral.ai/v1`                              | `codestral-latest`                               |
| SambaNova           | `https://api.sambanova.ai/v1`                            | `DeepSeek-V3.2` or `gpt-oss-120b`                |
| Hugging Face Router | `https://router.huggingface.co/v1`                       | `openai/gpt-oss-120b` or provider-suffixed model |
| NVIDIA NIM          | `https://integrate.api.nvidia.com/v1`                    | Copy exact model id from `build.nvidia.com`      |
| Vercel AI Gateway   | `https://ai-gateway.vercel.sh/v1`                        | Copy model id from Vercel model catalog          |
| Fireworks           | `https://api.fireworks.ai/inference/v1`                  | Copy serverless model id from Fireworks app      |
| Alibaba Qwen        | `https://dashscope-intl.aliyuncs.com/compatible-mode/v1` | Copy Qwen/Qwen-Coder id from Model Studio        |
| GitHub Models       | `https://models.github.ai/inference`                     | Copy model id from Marketplace “Code” tab        |

### 2. Google Gemini API

```bash
export GEMINI_API_KEY="YOUR_GEMINI_KEY"

curl "https://generativelanguage.googleapis.com/v1beta/models/gemini-3.1-pro-preview:generateContent?key=$GEMINI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "parts": [
          {
            "text": "Review this Python function for bugs, then rewrite it with tests:\n\n..."
          }
        ]
      }
    ]
  }'
```

## What I’d actually use first

For **maximum coding power**, start with **Gemini 3.1 Pro Preview**, especially the custom-tools variant when building agents. For the best “free model buffet,” use **OpenRouter** and try `qwen/qwen3-coder:free`, `minimax/minimax-m2.5:free`, and `inclusionai/ling-2.6-1t:free`. For **throughput**, use **Cerebras** first, then **Groq**. For serverless app integration with a daily free bucket, use **Cloudflare Workers AI**. For quick GitHub-native experiments, use **GitHub Models**, but expect very low daily caps on the strongest models.
