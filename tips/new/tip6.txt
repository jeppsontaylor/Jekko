As of **May 6, 2026**, the strongest “free” hosted coding LLM APIs are split into two buckets:

1. **Ongoing free/free-allocation APIs**: Google Gemini, Cloudflare Workers AI, OpenRouter free routes, Cerebras, Groq, SambaNova, Cohere trial, Mistral Experiment, GitHub Models, Hugging Face monthly credits, NVIDIA NIM dev APIs.
2. **Free but limited/temporary**: some OpenRouter free models are excellent but scheduled to disappear within days; some providers publish no exact RPM/RPH and only show limits in the dashboard.

I’m treating your “connections/hour” as **requests/hour (RPH)**, because LLM APIs usually publish **RPM, RPD, TPM/TPD**, not TCP connection counts. Where a provider publishes RPM but not RPH, I compute **RPH = RPM × 60**, but the **daily request cap or token cap can throttle you first**.

## Setup instructions and example API calls

### 1. Google Gemini API

Go to **[https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)**, create or view a Gemini API key, then set `GEMINI_API_KEY`. Google’s docs say Gemini API keys are created and managed in Google AI Studio. ([Google AI for Developers][16])

````bash
export GEMINI_API_KEY="YOUR_KEY"

curl "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-pro:generateContent?key=$GEMINI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "parts": [
          {
            "text": "You are a senior Python engineer. Find bugs and improve this function:\n\n```python\ndef fib(n):\n    if n < 2: return n\n    return fib(n-1)+fib(n-2)\n```"
          }
        ]
      }
    ]
  }'
````

Swap model names: `gemini-2.5-pro`, `gemini-3-flash-preview`, `gemini-2.5-flash`.

### 2. Cloudflare Workers AI

Create a Cloudflare account, open Workers AI in the dashboard, choose **Use REST API**, create a Workers AI API token, and copy your Account ID. Cloudflare’s docs specify that custom tokens need `Workers AI - Read` and `Workers AI - Edit`. ([Cloudflare Docs][17])

```bash
export CLOUDFLARE_ACCOUNT_ID="YOUR_ACCOUNT_ID"
export CLOUDFLARE_AUTH_TOKEN="YOUR_API_TOKEN"

curl "https://api.cloudflare.com/client/v4/accounts/$CLOUDFLARE_ACCOUNT_ID/ai/v1/chat/completions" \
  -H "Authorization: Bearer $CLOUDFLARE_AUTH_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "@cf/moonshotai/kimi-k2.6",
    "messages": [
      {
        "role": "user",
        "content": "Build a FastAPI endpoint with pytest tests and explain the design."
      }
    ]
  }'
```

Good model swaps:

```text
@cf/moonshotai/kimi-k2.6
@cf/nvidia/nemotron-3-120b-a12b
@cf/openai/gpt-oss-120b
@cf/qwen/qwen2.5-coder-32b-instruct
```

Cloudflare also supports OpenAI-compatible `/v1/chat/completions`, so OpenAI SDK-style clients work by changing the base URL and model name. ([Cloudflare Docs][18])

### 3. OpenRouter

Create a key at **[https://openrouter.ai/settings/keys](https://openrouter.ai/settings/keys)**. OpenRouter uses Bearer tokens and the OpenAI-compatible base URL `https://openrouter.ai/api/v1`. ([OpenRouter][19])

```bash
export OPENROUTER_API_KEY="YOUR_KEY"

curl https://openrouter.ai/api/v1/chat/completions \
  -H "Authorization: Bearer $OPENROUTER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "qwen/qwen3-coder:free",
    "messages": [
      {
        "role": "user",
        "content": "Refactor this TypeScript function for clarity and add edge-case tests."
      }
    ]
  }'
```

Good model swaps:

```text
qwen/qwen3-coder:free
tencent/hy3-preview:free
inclusionai/ling-2.6-1t:free
```

Strict free usage: `:free` model variants are capped at **20 requests/minute**, and accounts with less than 10 purchased credits are capped at **50 free-model requests/day**. ([OpenRouter][20])

### 4. Cerebras

Create a free API key at **[https://cloud.cerebras.ai/](https://cloud.cerebras.ai/)**, then set `CEREBRAS_API_KEY`. Cerebras’ quickstart explicitly links to a free API key and shows `CEREBRAS_API_KEY` setup. ([Cerebras Inference][21])

```bash
export CEREBRAS_API_KEY="YOUR_KEY"

curl https://api.cerebras.ai/v1/chat/completions \
  -H "Authorization: Bearer $CEREBRAS_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-oss-120b",
    "messages": [
      {
        "role": "user",
        "content": "Find the bug in this Rust function and propose a minimal patch."
      }
    ],
    "max_completion_tokens": 1024
  }'
```

Good model swaps:

```text
gpt-oss-120b
qwen-3-235b-a22b-instruct-2507
zai-glm-4.7
```

```bash
export SAMBANOVA_API_KEY="YOUR_KEY"
export SAMBANOVA_BASE_URL="YOUR_SAMBANOVA_BASE_URL_FROM_DASHBOARD"

curl "$SAMBANOVA_BASE_URL/chat/completions" \
```

### 7. Cohere

Create an account, then copy the automatically created trial key from **[https://dashboard.cohere.com/api-keys](https://dashboard.cohere.com/api-keys)**. Cohere states trial/evaluation keys are free but limited. ([Cohere Documentation][23])

```bash
export COHERE_API_KEY="YOUR_KEY"

curl https://api.cohere.com/v2/chat \
  -H "Authorization: Bearer $COHERE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "command-a-reasoning-08-2025",
    "messages": [
      {
        "role": "user",
        "content": "Design a safe database migration plan for a Django app with zero downtime."
      }
    ]
  }'
```

Cohere’s trial Chat API is **20 requests/minute**, and newer/trial model variants are limited to **1,000 API calls/month**. ([Cohere Documentation][23])

### 8. Mistral

Create an API key at **[https://console.mistral.ai/api-keys](https://console.mistral.ai/api-keys)**. Mistral says API keys authenticate requests, can be created in Studio, and that you can choose the free **Experiment** tier with conservative rate limits. ([Mistral AI][24])

```bash
export MISTRAL_API_KEY="YOUR_KEY"

curl https://api.mistral.ai/v1/chat/completions \
  -H "Authorization: Bearer $MISTRAL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "codestral-latest",
    "messages": [
      {
        "role": "user",
        "content": "Complete this parser and explain the edge cases."
      }
    ]
  }'
```

For fill-in-the-middle code completion:

```bash
curl https://api.mistral.ai/v1/fim/completions \
  -H "Authorization: Bearer $MISTRAL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "codestral-latest",
    "prompt": "def fibonacci(n: int):",
    "suffix": "n = int(input())\nprint(fibonacci(n))",
    "temperature": 0
  }'
```

Mistral’s coding docs say Codestral is trained for code generation and FIM completion and supports `codestral-latest` on both FIM and chat endpoints. ([Mistral AI][14])

### 9. GitHub Models

Use your GitHub account. For API access, create a Personal Access Token at **[https://github.com/settings/tokens](https://github.com/settings/tokens)** with `models:read` / `models` scope, then call GitHub Models. GitHub says free API usage is in public preview and uses GitHub credentials. ([GitHub Docs][25])

```bash
export GITHUB_TOKEN="YOUR_GITHUB_PAT_WITH_MODELS_READ"

curl -L https://models.github.ai/inference/chat/completions \
  -H "Accept: application/vnd.github+json" \
  -H "Authorization: Bearer $GITHUB_TOKEN" \
  -H "X-GitHub-Api-Version: 2022-11-28" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "deepseek/DeepSeek-R1-0528",
    "messages": [
      {
        "role": "user",
        "content": "Analyze this failing test and suggest the smallest code fix."
      }
    ]
  }'
```

To list exact model IDs available to your account:

```bash
curl -L \
  -H "Accept: application/vnd.github+json" \
  -H "Authorization: Bearer $GITHUB_TOKEN" \
  -H "X-GitHub-Api-Version: 2026-03-10" \
  https://models.github.ai/catalog/models
```

GitHub’s free API limits vary by model tier; for DeepSeek-R1 / DeepSeek-R1-0528 on Copilot Free, the table shows **1 RPM, 8 RPD, 4k input / 4k output tokens, concurrency 1**. ([GitHub Docs][25])

### 10. Hugging Face Inference Providers

Create a Hugging Face token at **[https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)** with **Make calls to Inference Providers** permission. Hugging Face supports an OpenAI-compatible router at `https://router.huggingface.co/v1`. ([Hugging Face][26])

```bash
export HF_TOKEN="YOUR_HF_TOKEN"

curl https://router.huggingface.co/v1/chat/completions \
  -H "Authorization: Bearer $HF_TOKEN" \
  -H "Content-Type: application/json" \

      }
    ]
  }'
```

Hugging Face’s free tier is best for testing, not volume: free users receive **$0.10 monthly credits**, and extra usage requires purchased credits. ([Hugging Face][27])

### 11. NVIDIA NIM APIs

Go to **[https://build.nvidia.com/](https://build.nvidia.com/)**, sign in, choose a model, and click **Get API Key** / **Try API**. NVIDIA advertises “Free serverless APIs for development,” but I did not find a public stable RPM/RPH table for the hosted dev APIs. ([NVIDIA NIM APIs][6])

Typical OpenAI-compatible shape, using the model-specific endpoint and slug shown in NVIDIA’s generated snippet:

```bash
export NVIDIA_API_KEY="YOUR_NVIDIA_KEY"

curl https://integrate.api.nvidia.com/v1/chat/completions \
  -H "Authorization: Bearer $NVIDIA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nvidia/nemotron-3-super",
    "messages": [

        "role": "user",

```

Verify the exact model slug in the NVIDIA “Try API” panel; the docs show Nemotron 3 Super examples using an OpenAI-compatible client and `nvidia/nemotron-3-super` for serving backends. ([NVIDIA API Documentation][28])

## My practical recommendation

For **best raw coding quality**, start with **Gemini 2.5 Pro**, **Cloudflare Kimi K2.6**, and **OpenRouter Qwen3 Coder / Hy3 / Ling**. For **highest free request throughput**, use **Cloudflare Workers AI** first, then **Cerebras** and **Groq**. For **code completion/FIM**, keep **Mistral Codestral** in the stack. For **occasional reasoning reviews**, add **Cohere Command A Reasoning**. For **experiments inside GitHub workflows**, GitHub Models is convenient but very low-volume.

One provider I would **not** include in a free ranking is **Together AI**: its own docs say it does not currently offer free trials and requires a minimum **$5 credit purchase** before platform access. ([docs.together.ai][29])
