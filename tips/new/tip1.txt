**Vercel AI Gateway, Z.ai, Kilo Gateway, Hugging Face Inference Providers, OVHcloud AI Endpoints, Cohere, and LLM7.**

I am **not** counting Together AI direct, DeepInfra direct, or iFlow direct: Together’s current docs say there is no free trial and a minimum $5 credit purchase is required; DeepInfra says you must add a card or prepay before use; iFlow CLI officially shut down on **April 17, 2026**. ## Provider setup and example API calls

### 1. Vercel AI Gateway

Use this for rows **1–7 and 9**.

Key URL: [https://vercel.com/ai-gateway](https://vercel.com/ai-gateway)
Env var: `AI_GATEWAY_API_KEY`
Free limit: **$5/month credits**; no Vercel-added fixed RPM/RPH, but upstreams may throttle. ([Vercel][18])

```bash
export AI_GATEWAY_API_KEY="YOUR_VERCEL_AI_GATEWAY_KEY"
export MODEL="anthropic/claude-sonnet-4.6"

curl https://ai-gateway.vercel.sh/v1/chat/completions \
  -H "Authorization: Bearer $AI_GATEWAY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "'"$MODEL"'",
    "messages": [
      {
        "role": "system",
        "content": "You are a senior software engineer. Return correct, production-quality code."
      },
      {
        "role": "user",
        "content": "Implement an LRU cache in TypeScript with O(1) get/put and include tests."
      }
    ],
    "temperature": 0.2
  }'
```

Good model swaps:

```bash
export MODEL="anthropic/claude-opus-4.7"
export MODEL="anthropic/claude-opus-4.6"
export MODEL="anthropic/claude-sonnet-4.6"
export MODEL="anthropic/claude-sonnet-4.5"
export MODEL="anthropic/claude-opus-4"
export MODEL="moonshotai/kimi-k2.6"
export MODEL="openai/gpt-5.4"
export MODEL="xai/grok-code-fast-1"
```

### 3. Kilo Gateway

Use this for rows **14–18**.

Key URL: [https://app.kilo.ai](https://app.kilo.ai)
Env var: `KILO_API_KEY`
Free limit: **200 requests/hour/IP** for free models. Kilo also allows anonymous access to free models, but authenticated use is cleaner for tooling. ([Kilo][14])

```bash
export KILO_API_KEY="YOUR_KILO_KEY"
export MODEL="x-ai/grok-code-fast-1:optimized:free"

curl https://api.kilo.ai/api/gateway/chat/completions \
  -H "Authorization: Bearer $KILO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "'"$MODEL"'",
    "messages": [
      {
        "role": "system",
        "content": "You are an expert coding assistant."
      },
      {
        "role": "user",
        "content": "Review this Rust function for lifetime bugs and return a corrected version."
      }
    ],
    "temperature": 0.2
  }'
```

Good free model swaps:

```bash
export MODEL="x-ai/grok-code-fast-1:optimized:free"
export MODEL="z-ai/glm-5:free"
export MODEL="minimax/minimax-m2.1:free"
export MODEL="arcee-ai/trinity-large-thinking:free"
export MODEL="kilo-auto/free"
```

You can list current Kilo models here:

```bash
curl https://api.kilo.ai/api/gateway/models
```

Kilo says free model availability and upstream limits can change, so check this endpoint before hardcoding a long-term router. ([Kilo][13])

### 4. Hugging Face Inference Providers

Use this for row **8**.

Key URL: [https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)
Env var: `HF_TOKEN`
Free limit: **$0.10/month inference credits** for free users; no single fixed RPM/RPH because Hugging Face routes to third-party providers. ([Hugging Face][9])

```bash
pip install -U huggingface_hub
export HF_TOKEN="YOUR_HF_TOKEN"
```

```python
import os
from huggingface_hub import InferenceClient

client = InferenceClient(
    provider="novita",
    api_key=os.environ["HF_TOKEN"],
)

completion = client.chat.completions.create(
    model="deepseek-ai/DeepSeek-V4-Pro",
    messages=[
        {
            "role": "system",
            "content": "You are a senior software engineer. Be precise and write working code.",
        },
        {
            "role": "user",
            "content": "Implement a Go worker pool with cancellation, backpressure, and tests.",
        },
    ],
    temperature=0.2,
)

print(completion.choices[0].message.content)
```

The free HF credit is small, so use it for “hardest prompt of the day” jobs rather than bulk code generation.

### 5. OVHcloud AI Endpoints

Use this for row **19**.

Key setup: OVH Public Cloud → AI Endpoints → API access key
Docs: [https://help.ovhcloud.com/csm/en-ie-public-cloud-ai-endpoints-getting-started](https://help.ovhcloud.com/csm/en-ie-public-cloud-ai-endpoints-getting-started)
Env vars: `OVH_AI_ENDPOINT_URL`, `OVH_AI_ENDPOINT_API_KEY`
Free/trial limits: anonymous access is **2 RPM / 120 RPH** per IP/model; authenticated API key access is **400 RPM / 24,000 RPH** per project/model. OVH also advertises Public Cloud free trial credit, so authenticated use may be free until credits run out. ([OVH Cloud Help][16])

OVH endpoints are model-specific, so copy the model’s OpenAI-compatible URL from the OVH AI Endpoints catalog.

```bash
export OVH_AI_ENDPOINT_URL="PASTE_THE_MODEL_ENDPOINT_URL_FROM_OVH"
export OVH_AI_ENDPOINT_API_KEY="YOUR_OVH_AI_ENDPOINT_KEY"

curl "$OVH_AI_ENDPOINT_URL/v1/chat/completions" \
  -H "Authorization: Bearer $OVH_AI_ENDPOINT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "Qwen3-Coder-30B-A3B-Instruct",
    "messages": [
      {
        "role": "user",
        "content": "Write a secure FastAPI file-upload endpoint with size limits, MIME checks, and pytest tests."
      }
    ],
    "temperature": 0.2
  }'
```

## Best missing keys to add first

Start with these in order:

1. **Vercel AI Gateway** — best missed key by far because it gives free monthly credits over many top proprietary and open models.
2. **Z.ai** — strong GLM coding/agent models, including free/limited-time-free options.
3. **Kilo Gateway** — true free model routes with a clear **200 RPH/IP** cap.
4. **Hugging Face Inference Providers** — tiny free credit, but useful for one or two high-value calls to models like DeepSeek V4 Pro.
5. **OVHcloud AI Endpoints** — strong authenticated throughput if you use the free trial credit.
6. **Cohere** — good reasoning fallback with clear free monthly call limits.
