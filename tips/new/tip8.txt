## Ranking assumptions

I treated “free” as **hosted online API access you can start at $0**, but not all free tiers are equal. Some are recurring free tiers, some are preview/evaluation tiers, and some are “first N tokens free, then paid.” For coding strength, I weighted public coding-benchmark signals, coding-specific model cards, context length, reasoning/tool-use claims, and whether the model is actually available on a free hosted API. One public coding signal I used was LiveCodeBench, where recent top free-access candidates include Gemini 3 Flash Preview, GPT-OSS-120B, Kimi K2, Gemini 2.5 Pro, Qwen3-235B, and DeepSeek-R1-0528; I excluded models that look powerful but whose official API page shows no free tier for that model. ([Price Per Token][1])

“RPH” below means **requests per hour**. When a provider publishes RPM but not RPH, I show the theoretical `RPM × 60`, but the **daily cap** often stops you first.

## Setup URLs and API-key instructions

### 1. Google Gemini API

Setup URL: `https://aistudio.google.com/apikey`

Steps:

1. Sign in with a Google account.
2. Open **Get API key**.
3. Create an API key for a Google Cloud project.
4. Store it as an environment variable.

```bash
export GEMINI_API_KEY="paste_key_here"
```

Example call:

```bash
curl "https://generativelanguage.googleapis.com/v1beta/models/gemini-3-flash-preview:generateContent?key=$GEMINI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "parts": [
          {
            "text": "Write a clean Python function that topologically sorts a directed acyclic graph. Include tests."
          }
        ]
      }
    ]
  }'
```

Google’s free tier is real, but exact request limits are not stable public constants; the official docs say limits vary by model, project, tier, and capacity, and should be checked in AI Studio. ([Google AI for Developers][2])

### 2. GitHub Models

Setup URLs:

* `https://github.com/settings/tokens`
* `https://github.com/marketplace/models`

Steps:

1. Create or open a GitHub account.
2. Go to personal access tokens.
3. Create a token with the **models** permission/scope.
4. Store it as `GITHUB_TOKEN`.

```bash
export GITHUB_TOKEN="paste_token_here"
```

Example call:

```bash
curl "https://models.github.ai/inference/chat/completions" \
  -H "Authorization: Bearer $GITHUB_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/gpt-4.1",
    "messages": [
      {
        "role": "user",
        "content": "Review this TypeScript function for bugs and rewrite it with strict types."
      }
    ]
  }'
```

GitHub Models has no-cost rate-limited access for GitHub accounts, but once your free quota is hit, usage is blocked unless paid usage is enabled. ([GitHub Docs][3])

### 3. OpenRouter free models

Setup URL: `https://openrouter.ai/keys`

Steps:

1. Create an OpenRouter account.
2. Open the API keys page.
3. Create a key.
4. Use models ending in `:free`.

```bash
export OPENROUTER_API_KEY="paste_key_here"
```

Example call:

```bash
curl "https://openrouter.ai/api/v1/chat/completions" \
  -H "Authorization: Bearer $OPENROUTER_API_KEY" \
  -H "Content-Type: application/json" \
  -H "HTTP-Referer: https://example.com" \
  -H "X-Title: coding-test" \
  -d '{
    "model": "qwen/qwen3-coder:free",
    "messages": [
      {
        "role": "user",
        "content": "Implement a Rust LRU cache with O(1) get/put and explain the tradeoffs."
      }
    ]
  }'
```

OpenRouter free models are explicitly marked with `:free`; the published free limit is 20 requests per minute, with a daily free-model cap of 50 requests unless the account has at least $10 credits, in which case the cap rises to 1,000 daily requests. ([OpenRouter][14])

### 6. Cloudflare Workers AI

Setup URL: `https://dash.cloudflare.com/`

Steps:

1. Create a Cloudflare account.
2. Go to **Workers AI**.
3. Create an API token with Workers AI read/edit permissions.
4. Copy your Cloudflare Account ID.
5. Store both values.

```bash
export CLOUDFLARE_API_TOKEN="paste_token_here"
export CLOUDFLARE_ACCOUNT_ID="paste_account_id_here"
```

Example call:

```bash
curl "https://api.cloudflare.com/client/v4/accounts/$CLOUDFLARE_ACCOUNT_ID/ai/run/@cf/moonshotai/kimi-k2.6" \
  -H "Authorization: Bearer $CLOUDFLARE_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {
        "role": "user",
        "content": "Generate a production-ready FastAPI CRUD service with Pydantic models and pytest tests."
      }
    ]
  }'
```

Cloudflare’s free allocation is 10,000 Neurons per day, resetting at 00:00 UTC, and most text-generation models are limited to 300 RPM. ([Cloudflare Docs][9])

### 9. Hugging Face Inference Providers

Setup URL: `https://huggingface.co/settings/tokens`

Steps:

1. Create a Hugging Face account.
2. Create a token with permission to call Inference Providers.
3. Install the client or use the OpenAI-compatible endpoint.
4. Store the token as `HF_TOKEN`.

```bash
export HF_TOKEN="paste_token_here"
pip install huggingface_hub
```

Example Python call:

```python
import os
from huggingface_hub import InferenceClient

client = InferenceClient(token=os.environ["HF_TOKEN"])

response = client.chat.completions.create(
    model="deepseek-ai/DeepSeek-V3-0324",
    messages=[
        {
            "role": "user",
            "content": "Write a memory-safe C++17 parser for a simple arithmetic grammar."
        }
    ],
)

print(response.choices[0].message.content)
```

Hugging Face provides monthly free inference credits, but the useful limit is credit-based rather than a simple public RPM/RPH for every model/provider combination. Free users get a small monthly credit allocation, and Pro users get more. ([Hugging Face][17])

## Best practical picks

For **raw coding power**, start with **Gemini 3 Flash Preview**, **Gemini 2.5 Pro**, **GitHub GPT-4.1**, **OpenRouter Qwen3-Coder**, and **OpenRouter Kimi K2**.

For **best free bandwidth**, start with **Cloudflare Workers AI**, **Scaleway**, **Cerebras**, and **Groq**. Cloudflare and Scaleway have the highest RPM-style limits, but Cloudflare is constrained by daily Neurons and Scaleway’s free tier converts to billing after the free token allocation.

For **agentic coding loops**, the best free stack is likely:
`Qwen3-Coder:free` on OpenRouter for hard repo reasoning, `gpt-oss-120b` or `qwen-3-235b` on Cerebras for fast iterations, and Cloudflare/Scaleway for high-volume cheaper passes.
