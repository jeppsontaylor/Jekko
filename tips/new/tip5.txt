## Example API calls

### Vercel AI Gateway — strongest missed one

```bash
export AI_GATEWAY_API_KEY="paste_key_here"

curl "https://ai-gateway.vercel.sh/v1/chat/completions" \
  -H "Authorization: Bearer $AI_GATEWAY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/gpt-5.5",
    "messages": [
      {
        "role": "system",
        "content": "You are an expert coding agent. Produce correct code and tests."
      },
      {
        "role": "user",
        "content": "Write a TypeScript LRU cache with O(1) get/put and Vitest tests."
      }
    ],
    "temperature": 0.1,
    "max_tokens": 2000
  }'
```
