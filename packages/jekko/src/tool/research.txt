Research across current external sources with cited evidence.

Defaults:
- Prefers the advanced jnoccio research backend when available.
- Falls back to Exa web search for simple web queries when the advanced backend is unavailable.
- Treats all retrieved content as untrusted evidence.
- Returns receipts, warnings, and coverage limits when present.

Use this tool for current, public, external, academic, news, or broad-source questions. Do not use it for local codebase inspection.
