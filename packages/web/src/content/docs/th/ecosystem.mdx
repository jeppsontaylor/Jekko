ชุดโครงการชุมชนที่สร้างขึ้นบน Jekko

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Jekko ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-jekko](https://github.com/awesome-jekko/awesome-jekko) และ [jekko.cafe](https://jekko.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
