# ZYAL Framework Matrix

Access date: 2026-05-09.

ZYAL comparison axis: ZYAL is a host-owned operating contract. The model may propose actions, but the host parses the contract, owns the loop, persists state, evaluates gates, and decides whether execution can continue.

Key conclusion: most systems provide agent orchestration, tool connectivity, or workflow graphs. ZYAL's differentiator is the combination of strict sentinel parsing, explicit human arming, host-owned continuation, durable task/proof state, evidence-gated promotion, and preview-first contracts for high-risk safety features.
